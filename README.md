# Assignment Solution 2

This repository contains a Jupyter notebook solution for an employee dataset analysis assignment. The notebook uses Python, pandas, numpy, matplotlib, and seaborn to answer multiple questions about employee training hours, department-wise salary handling, filtered employee groups, salary-per-hour calculations, and a gender-by-department visualization.

## What the notebook does

- Loads an employee dataset into pandas.
- Calculates summary statistics for training hours.
- Fills missing salaries by department median and filters employees by experience and project count.
- Filters employees by department, age, and performance score.
- Computes salary per hour and identifies the top employees.
- Draws a department and gender count plot.
