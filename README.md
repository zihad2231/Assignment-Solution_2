# Practice_Numpy-and-Pandas

Practice1.ipynb is a beginner-friendly Jupyter Notebook for practicing NumPy and pandas basics. It focuses on loading tabular data, exploring the structure of a DataFrame, and reviewing sample rows so you can get comfortable with the core workflow used in early data analysis exercises.

## What It Covers
- Loading data into a pandas DataFrame
- Inspecting rows, columns, and basic structure
- Practicing simple exploratory data analysis steps
- Building confidence with NumPy and pandas in Jupyter Notebook

## Technologies Used
- Python
- pandas
- NumPy
- Jupyter Notebook
